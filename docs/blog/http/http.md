浏览器的http请求过程
1.输入url开始跳转(redirect)
2.查看缓存(app cache)
3.DNS查找，域名
4.创建安全链接和TCP链接
5.开始发送http request
6.接受response

## 网络协议分层
- 应用层
- 传输层
- 网络层 数据在结点之间传输创建逻辑链路（电脑访问服务器）
- 数据链路层 通信实体间建立数据链路链接（硬件和网路的链接）
- 物理层 定义物理设备如何传输数据（硬件：网线、光缆...）
